# Privacy Policy Content for Ai Shepherd

## App Info
- App Name: Ai牧者 (Ai Shepherd)
- Developer: Ai Shepherd Team
- Contact: support@aishepherd.app (placeholder)

- Last Updated: February 27, 2026

## Key Facts
- No account required for core features (Bible reading, AI chat, Daily Verse)
- Optional account registration via email/password, Apple Sign In, or Google Sign In
- AI responses generated by third-party LLM services
- Push notifications (optional, user-controlled)
- No advertising, no selling of user data
- Data stored on secure servers

## Data Collected
1. Account data (if registered): email, name, login method
2. Usage data: daily verse history, saved verses, verse notes, prayer journal (stored locally or on server if logged in)
3. AI conversation history (stored on server if logged in, local only if not logged in)
4. Device info: push notification token (if notifications enabled)
5. Language and theme preferences

## Third-Party Services
- OpenAI / LLM provider: AI response generation
- Azure Cognitive Services: Text-to-speech
- Expo: Push notification delivery
- Apple Sign In / Google Sign In: Authentication
- Stripe: Subscription payments (if applicable)

## User Rights
- Access, correct, or delete account data at any time
- Disable push notifications via device settings
- Delete account via Settings > Account
